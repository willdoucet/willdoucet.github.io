Summary: Latitude vs. X

The purpose of this project was to analyze the effect of latitude on climate using data on Temperature, Humidity, Cloudiness, and Wind Speed. The data was collected from the OpenWeatherMap API for 500 cities across the world of varying latitudes.

Matplotlib was used to graph the data.
